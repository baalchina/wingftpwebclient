Using Text Editor

The Text Editor is used to view or edit text files in the Web Client. It supports all kinds of text files, such as .txt, .xml, .java... You can open the Text Editor by double-clicking on the common text files or clicking the toolbar button "Text Edit" after a file has been selected.

 

 

To find specific characters or words

•Open the Text Editor.
•Click the button "search" on the top left, or use the hotkey "ctrl+F".
•In "search" field, type the characters or words you want to find.
•Click the button "find next".

 

 

To find and replace specific characters or words

•Click the button "search" on the top left, or use the hotkey "ctrl+F".
•In "search" field, type the characters or words you want to find.
•In "replace" field, type the replacement text.
•To replace all instances of the text at once, click "replace all". Or, to replace one instance at a time, click "find next", and then click "replace".

 

 

To go to a specific line

•Click the button "go to line" , or use the hotkey "ctrl+G".
•Type the line number you want the cursor to jump to.

 

 

Undo

•Click the button "undo" , or use the hotkey "ctrl+Z".

 

 

Redo

•Click the button "redo" , or use the hotkey "ctrl+Y".

 

 

To change the font size

•Click the drop-down list box ,  and select the Font size you want.

 

 

Save the change

•Click the button "save" on the top left, or use the hotkey "ctrl+S".
